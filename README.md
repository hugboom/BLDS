## Bone Lesion Detection System (BLDS)
This is the official repository for `A clinically applicable AI system for the detection and diagnosis of bone metastases using CT scans`. It uses the [FPN](https://arxiv.org/abs/1612.03144) network to detect 3D bone lesion bounding boxes in CT images.
